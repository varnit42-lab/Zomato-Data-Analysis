# Zomato-Data-Analysis
This project analyzes a Zomato restaurant dataset using *Python* and *Pandas*. The objective is to perform data cleaning, explore the dataset, and answer business-related questions by extracting meaningful insights.

## 📂 Dataset
- Zomato Restaurant Dataset (CSV)
- Contains information such as:
  - Restaurant Name
  - Ratings
  - Votes
  - Online Ordering
  - Table Booking
  - Approximate Cost for Two
  - Listed Restaurant Type

## 🛠️ Technologies Used
- Python
- Pandas
- Jupyter Notebook

## 📊 Analysis Performed
1. Data Cleaning
2. Dataset Information
3. Descriptive Statistics
4. Number of restaurants offering online orders
5. Average cost for two people
6. Number of restaurants with ratings above 4
7. Highest number of votes received by a restaurant
8. Average rating for each restaurant type
9. Summary of key insights

## 📈 Key Findings
- 58 restaurants offer online ordering.
- The average cost for two people is approximately ₹418.
- Only 34 restaurants have ratings of 4 or above.
- The highest number of votes received by a restaurant is 4884.
- The "Other" category has the highest average rating, while "Dining" has the lowest average rating.
